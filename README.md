# BuildRight Construction Website

BuildRight Construction is a simple static website for a fictional construction company. The project presents the company as a trusted building partner for residential construction, commercial building, renovations, and project management.

The website is built with plain HTML and CSS, so it does not require a framework, build tool, or package installation. It can be opened directly in a browser.

## Project Overview

The project contains a two-page website:

- `index.html` - the main homepage with the hero section, company introduction, services, recent projects, about summary, and contact information.
- `about.html` - a detailed about page with the company story, mission, vision, values, leadership team, achievements, reasons to choose the company, and client testimonials.
- `style.css` - the shared stylesheet used by both pages.

## Main Features

- Sticky header with navigation links
- Homepage hero section with a call-to-action button
- Services section for residential, commercial, renovation, and project management work
- Recent projects section
- Contact section with phone, email, and address details
- Dedicated about page with company history and values
- Leadership team section
- Achievements and testimonials sections
- Responsive layout for smaller screens
- Consistent green construction-themed design

## Pages

### Home Page

The homepage introduces BuildRight Construction and gives visitors a quick overview of the company. It includes:

- A hero message: "Building Tomorrow's Structures Today"
- Company mission, vision, and values
- Service cards
- Recent project examples
- A short about section
- Contact details

### About Page

The about page gives more detail about the fictional company. It explains that BuildRight was founded in 2006 and focuses on quality, safety, sustainability, and strong client relationships.

This page also includes:

- Company story
- Mission, vision, and core values
- Leadership team profiles
- Company achievements
- Reasons to choose BuildRight
- Client testimonials

## Technologies Used

- HTML5
- CSS3
- Responsive CSS grid layouts
- Media queries for mobile screens

## How to Run the Project

Because this is a static website, you can run it by opening the HTML files directly in your browser.

1. Open the project folder.
2. Double-click `index.html`.
3. Use the navigation menu to move between the homepage and the about page.

You can also use the VS Code Code Runner settings for HTML files to open the current page in the browser.

## Purpose of the Project

This project is useful for practicing the basics of building a complete business website with HTML and CSS. It demonstrates page structure, navigation, reusable styling, responsive grids, sections, cards, hover effects, and consistent branding across multiple pages.
